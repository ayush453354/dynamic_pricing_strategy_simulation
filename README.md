# Dynamic Pricing Strategy Simulation

## Objective
To optimize pricing strategy using demand elasticity modeling and profit maximization.

## Approach
- Customer segmentation (Sensitive, Moderate, Premium)
- Willingness-to-pay simulation
- Revenue and profit modeling
- Sensitivity analysis

## Results
Identified optimal pricing point improving projected profit under multiple demand scenarios.

## Tools Used
- Python
- NumPy
- Pandas
- Matplotlib
